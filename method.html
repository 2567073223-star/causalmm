理论基础

后门调整准则

视觉注意力 Aᵢ 对输出 O 的因果效应通过后门调整计算，消除视觉先验 Pᵥ 的混淆：

后门调整公式
P(O | do(Aᵢ = a)) = Σpᵥ P(O | Aᵢ = a, Pᵥ = pᵥ) · P(Pᵥ = pᵥ)
核心机制

四种注意力干预方式

🎲

随机注意力

A* = random(A)

均匀随机分布替换原始权重

➗

均匀注意力

A* = uniform(H·W)

注意力均分到所有空间位置

🔄

反转注意力

A* = max(A) - A

反转权重，突出被忽视区域

🔀

扰乱注意力

A* = shuffle(A)

随机打乱注意力空间顺序

反事实模式

三种推理模式

👁️

仅视觉反事实

do(Aᵢ = Aᵢ*)，仅在视觉编码器层干预，适合图像质量高的场景

do(Aᵢ)
💬

仅语言反事实

do(Aₜ = Aₜ*)，仅在 LLM 自注意力层干预，适合语言偏见主导的场景

do(Aₜ)
🔗

多模态联合反事实

do(Aᵢ = Aᵢ*, Aₜ = Aₜ*)，两层同时干预，效果最优，推荐默认使用

do(Aᵢ, Aₜ)
因果解码

因果修正解码

最终 token 生成概率由因果修正 logits 计算：

因果修正解码公式
ℓnext = ℓt + γ · ((ℓt − ℓcfv,t) + (ℓt − ℓcfl,t))

其中 ℓt 为原始 logits，ℓcfv,t / ℓcfl,t 为视觉/语言反事实 logits，γ 为因果效应强度超参数（论文默认 γ=1.0）。
